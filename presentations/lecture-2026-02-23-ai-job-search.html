Что такое Claude Code

  • CLI-инструмент от Anthropic
  • AI-ассистент прямо в терминале
  • Читает файлы, пишет код, выполняет команды
  • Работает с любыми проектами и задачами
  • Расширяется через MCP-серверы
$ claude

  > Найди 5 вакансий Product Manager
    в Москве от 300к и подготовь
    персонализированные отклики

  Claude: Ищу вакансии на hh.ru...
  ✓ Найдено 12 позиций
  ✓ Отфильтровано по зарплате: 5
  ✓ Генерирую сопроводительные...

Как установить

  1. Установить Node.js — nodejs.org (LTS версия)
  2. Открыть терминал и выполнить: npm install -g @anthropic-ai/claude-code
  3. Запустить: claude
  4. Авторизоваться через claude.ai (Pro $20/мес или Max $100/мес)
Альтернатива — через brew (macOS) brew install node
npm install -g @anthropic-ai/claude-code

Блок 2

AI-агент ≠ AI-чат

Чат (ChatGPT, Claude.ai)

  • Вопрос → ответ, одноразово
  • Не запоминает контекст между сессиями
  • Не выполняет действий
  • Ты делаешь работу сам

Агент (Claude Code, AIHawk)

  • Задача → автономное выполнение
  • Помнит контекст, учится
  • Выполняет действия: файлы, API, браузер
  • Агент делает работу за тебя

Агент = AI + инструменты + автономность
