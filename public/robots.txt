# Sprint 2 fix v2 (2026-04-13): per robots.txt spec, Disallow rules under
# `User-agent: *` are NOT inherited by bots that have their own group.
# So we MUST repeat Disallow lines in each named bot block, or simply
# rely on the wildcard alone (most major bots respect *).
#
# Approach: removed all named "Allow: /" blocks (they were redundant — bots
# default to allowed). Single wildcard group with Disallow rules now governs
# ALL bots uniformly.

User-agent: *
Allow: /

# Block API + auth + analytics paths to free crawl budget for SSG content.
# These paths return JSON or auth pages — not useful for index.
Disallow: /v1/
Disallow: /stats/
Disallow: /auth/
Disallow: /admin/
